    Instruction For this project, you are required to develop a website. The subject and contents of this website is left open, and you can come up with your own idea. For instance, you may represent a place such as “Taj Mahal”, or represent a hotel, or represent a visiting place such as “New York”.
    What to prepare:
    Your website MUST meet the following requirements:

    • At least 10 pages
    • Responsive to mobile devices
    • Apply Cascading Style Sheet (CSS) in all pages
    • Apply JQuery or JavaScript to at least five pages
    • Use all the html tags discussed during the class
    • Follow all guidelines when designing webpages as discussed n class
    Once developed, you are requested to design and develop three different CSS files to be used by the website. Your CSS must be thoughtful and previously planned, e.g., use meaningful names for different classes, ids, etc. You need to use the CSS so that the whole website can be re-styled by using different set of CSS files. Our goal is to change the website design by simply replacing the “.css” files with the existing ones. So, developing a well-designed architecture for each page and appropriate style for that page is well demanding.
